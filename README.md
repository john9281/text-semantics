# text-semantics
解决文本语义相似的核心是让模型理解什么是语义。本文先介绍embedding之前有哪些文本相似方法，embedding之后文本相似的发展，进一步尝试对语义下定义，最后给出一个好的语义相似模型，应该具有哪些最基本能力。
## embedding前的文本相似
1 编辑距离

2 simhash

3 bm25

4 杰卡德相似系数

5 tf-idf

还有很多其他算法，这些算法有各自的优点，但同时都有共同的不足。如：句子一：序列标注；句子二：字符分类。这两句话是很相似的，用以上算法很难解。
